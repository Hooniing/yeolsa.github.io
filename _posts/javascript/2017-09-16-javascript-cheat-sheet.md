## 자바스크립트 치트키


### Math

- 배열에서 최대값 구하기: Math.max.apply(null, numArray);
- 배열에서 최소값 구하기: Math.min.apply(null, numArray);

### Array

- 배열에서 요소 제거하기: array.splice(startIndex, deleteCount);
- 배열 복사하기: Array.from(array);
